Bootloader Functions
USB Link (link)
Collaboration diagram for Bootloader Functions:

Detailed Description

These functions are supported when the device is in bootloader mode as well as normal mode. The functions link_readflash(), link_writeflash(), and link_erase_flash() are only available in bootloader mode.
