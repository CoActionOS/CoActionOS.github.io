Memory (RAM/Flash)

#include <mem.h>

Collaboration diagram for mem_attr_t:

Detailed Description

This defines the attributes of the memory.

It is used with I_MEM_GETATTR. The attributes are read-only so I_MEM_SETATTR has no effect.
